Domains

Full pipeline mode.

There are two ways the pipeline can be used in this mode:

1. Predicting protein features and building phylogenetic tree using full-length sequences.

If you want to use this option just paste your sequences or upload a file and click SUBMIT button.

2. Building phylogenetic tree using fragments of protein sequences and predicting protein features using full-length sequences.

If you want to use this option:

a) Click Add second area button. New query area will appear.

b) Put fragments of sequences into the first area (or choose a file). Protein sequences in this area will be used for alignment and building phylogentic tree.

c) Put full-length sequences into the second area (or choose a file). Protein features will be predicted using these sequences.

Partial pipeline mode.

If you have your own phylogenetic tree and protein sequences used to produce the tree use this mode:

a) In Domains section click Partial Pipeline button.

b) Upload protein sequences used to build the tree (Choose file with protein sequences button).

c) Optional, if you want the alignment to be ordered and enumerated according to the tree branches order, upload your alignment (Choose file with alignment (optional) button).

d) Upload the tree in newick format (Choose file with tree (Newick) button).

Neighborhoods

You can easily visualize gene neighborhoods on the phylogenetic tree running our neighborhoods analyzer.

Protein names in the file with the sequences or the tree should start with protein identifiers.

Suitable identifiers:

a) RefSeq Id

b) Locus tag, either old or new

b) MiST Id

The analyzer can be run in two ways:

1) by pasting protein sequences (or uploading corresponding file)

2) by pasting phylogenetic tree in newick format (or uploading corresponding file)
