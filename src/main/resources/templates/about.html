Full pipeline mode.

There are two ways the pipeline can be used in this mode:

1) Predicting protein features and building phylogenetic tree using full-length sequences.

If you want to use this option just paste your sequences or upload a file and click 'SUBMIT' button.

2) Building phylogenetic tree using fragments of protein sequences and predicting protein features using full-length sequences.

If you want to use this option:

a) Click "Add second area" button. New query area will appear.

b) Put full-length sequences into the first area (or choose a file). Protein sequences in this area will be used to predict features.

c) Put fragments of sequences into the second area (or choose a file). Protein sequences in this area will be used for alignment and building a tree.

Partial pipeline mode.

If you have your own phylogenetic tree and the protein sequences used to produce the tree use this mode:

a) Upload protein sequences used to build the tree ("Choose file with protein sequences" button).

b) Optional, if you want the alignment to be ordered and enumerated according to the tree branches order, upload your alignment ("Choose file with alignment (optional)" button).

c) Upload the tree in newick format ("Choose file with tree (Newick)" button).
